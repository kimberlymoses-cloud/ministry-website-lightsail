# ministry-website-lightsail
WordPress ministry site deployed on AWS Lightsail - migrated from legacy .NET
Migrated a legacy .NET ministry website to WordPress on AWS Lightsail. Built to share sermons, resources, and connect with community.

Live Site: https://prophetessk.com
Tech Stack: AWS Lightsail, WordPress, PHP, MySQL, Apache, SSL

Key Skills Demonstrated
Cloud infrastructure deployment on AWS Lightsail
Linux server administration via SSH
WordPress installation & configuration
DNS management + SSL certificate setup with Let's Encrypt
Data migration from legacy .NET application
Cost optimization: Free for the first 90 days and then running production site for $7/month
Architecture
User → Route 53/Domain → AWS Lightsail Instance → WordPress (Apache/PHP/MySQL)
